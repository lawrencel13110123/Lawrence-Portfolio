Overview

Hollywood Trivia is a Javascript application that features a series fo Hollywood related multiple choice questions. User has 10 seconds to answer each question. Upon submission, a gif along with the correct answer will be displayed. At the end of the game, user will see their total correct answer, wrong answer, and unanswered questions.
