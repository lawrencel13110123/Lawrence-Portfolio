Overview

Node Flashcard is a Node.js CLI application that utilizes the Inquirer npm package to prompt user a series of flashcard questions and answers. User also has the option to add custom questions after the first round of the game. The application has two versions: basic-card-game.js and cloze-card-game.js. The basic version has questions in this format: "Who is the first president of United States ?" The cloze version has questions in this format: "...is the first president of United States."
