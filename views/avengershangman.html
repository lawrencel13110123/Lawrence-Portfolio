Overview

Avengers Hangman is a Javascript app that uses document.onkeyup to register a user's click. The goal of the game is to figure out the word that matches the blank space. Each round, the user will be displayed a random word (blank spaces), a Marvel hero, and a Marvel interesting fact. User also has the choice to play the Marvel theme sound in the background.
