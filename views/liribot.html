Overview

Liri is like iPhone's Siri. However, while Siri is a Speech Interpretation and Recognition Interface, Liri is a Language Interpretation and Recognition Interface. Liri will be a command line Node.js app that takes in parameters and gives you back data. Users can give these commands:

  • node liri.js my-tweets
    • returns top 20 tweets from CNN
  • node liri.js spotify-this-song 'song name'
    • returns information on all the songs that matches the 'song name'
  • node liri.js movie-this 'movie name'
    • returns a movie that matches the 'movie name'
  • node liri.js do-what-it-says
    • runs the command in random.txt
    • presets to run spotify-this-song for "I Want it That Way"
